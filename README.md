# HTTP Response Inspector
Back-end sample that inspects an HTTP GET response for a public website.

# Instructions
1. Terminal 1: Run the application (port 5000 by default): `python app.py`
2. Terminal 2: Request a website in the GET parameter `url`: `curl "http://127.0.0.1:5000/inspect?url=https://www.ek.dk"`
3. Terminal 1: Check the output
